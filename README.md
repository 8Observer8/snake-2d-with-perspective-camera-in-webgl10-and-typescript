# snake-2d-with-perspective-camera-in-webgl10-and-typescript
Snake 2D With Perspective Camera in WebGL 1.0, TypeScript, glMatrix, AMD and RequireJS

- download or clone
- npm install
- npm run build-debug
- http-server
- open in browser: localhost:8080
